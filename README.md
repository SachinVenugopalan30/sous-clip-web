# sous-clip-web

Landing page for [Sous Clip](https://github.com/SachinVenugopalan30/sous-clip) - a self-hosted recipe extractor for cooking videos.

Built with [Astro](https://astro.build). Static site, zero backend.

## Dev

```sh
npm install
npm run dev       # localhost:4321
```

## Build

```sh
npm run build     # outputs to dist/
npm run preview   # preview the build
```

## Deploy

Drop the `dist/` folder on Vercel, Netlify, or any static host.
